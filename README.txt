相册圈人DEMO
介绍：
相册圈人DEMO是类似人人、QQ空间上的圈人功能，提供一张图片，可以在图片上圈定一个区域并标记区域名称，数据存储在mysql数据库中；
初始程序是从网上找的一个圈人demo，但是不能存储，不能同数据库交互。
本demo是基于该demo实现的一个完整的圈人程序。

版本：
1.0 增加圈人功能

环境：
linux+php+mysql+js

FILE:
index.html     //demo演示页面，js主程序页面
write.php	//ajax同数据库交互写入的php程序
read.php	//ajax读取数据库信息
ui.core.js
ui.draggable.js  
ui.resizable.js
